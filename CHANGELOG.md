# Changelog

## v1.0.0 — Initial Release

### Features
- Standalone Flutter app wrapping StableClaw AI Gateway for Android
- One-tap setup: downloads Ubuntu rootfs, Node.js 22, and StableClaw
- Built-in terminal emulator with extra keys toolbar
- Gateway start/stop with status monitoring
- AI provider configuration (7 providers)
- SSH remote access
- Node device capabilities (camera, flash, location, screen, sensor, haptic, serial)
- Web dashboard via WebView
- Optional packages: Go, Homebrew, OpenSSH
- Foreground service for background persistence
- No root required
